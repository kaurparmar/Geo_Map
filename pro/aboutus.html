About Us

Welcome to our Geo-Referenced Map Interface project! Our platform leverages open-source technologies to create an interactive and dynamic mapping experience.

Our Goal

Our mission is to build a user-friendly mapping tool that enables users to visualize geo-referenced data, add markers, and access location-based insights.

Technologies Used

  • Frontend: HTML, CSS, Tailwind CSS, Javascript, JQuery
  • Map Integration: Leaflet.js with OpenStreetMap
  • Backend: PHP for data handling
  • Database: MySQL

Why Choose Open Source?

We believe in the power of open-source technologies to develop cost-effective, scalable, and accessible solutions.
